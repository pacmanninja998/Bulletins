Getting Started

  1. Go to the Edit Bulletins page to add new bulletins
  2. Paste the bulletin text and enter the bulletin number
  3. Select whether it's for Engineer or Conductor position
  4. Review the automatically extracted information
  5. Save the bulletin to the system
